1
Get PayPal Credentials

Visit developer.paypal.com and sign in with your PayPal business account.

Navigate to Apps & Credentials and copy your:

  • Client ID
  • Client Secret
2
Update .env File

Edit the .env file in your project root:

PAYPAL_CLIENT_ID=YOUR_CLIENT_ID_HERE PAYPAL_SECRET=YOUR_SECRET_HERE PAYPAL_PAYOUT_EMAIL=your-business@paypal.com PAYPAL_ENV=sandbox

Replace with your actual credentials from PayPal Dashboard

3
Configure Email Delivery

For sending license keys via email, configure SMTP (Gmail recommended):

SMTP_HOST=smtp.gmail.com SMTP_PORT=587 SMTP_USER=your_email@gmail.com SMTP_PASS=your_16_char_app_password SMTP_FROM=noreply@yourdomain.com

üîß Troubleshooting

‚ùå "PayPal Not Configured" Error

  • Check PAYPAL_CLIENT_ID is set in .env
  • Make sure it doesn't contain "YOUR_" or placeholder text
  • Restart the server after changing .env
  • Check server logs for errors

‚ùå Checkout Page Shows "example.com"

  • Visit /server/config to verify client ID is being returned
  • Check browser console (F12) for error messages
  • Verify PAYPAL_CLIENT_ID is not empty in .env

‚ùå License Email Not Received

  • Check SMTP_USER and SMTP_PASS in .env
  • Gmail users: verify 16-character app password
  • Check spam/junk folder
  • Check server logs for email errors

‚ùå Payout Not Processing

  • Verify PAYPAL_PAYOUT_EMAIL is set correctly
  • Check PayPal API credentials are valid
  • Review server logs for payout errors
  • Ensure payout amount exceeds minimum threshold
